A robust **Bank Management System** built with C++ that demonstrates the practical application of **Functional Programming approach**.
The system provides an efficient way to manage client data through a console interface, with persistent storage using file handling.

##   Key Features
*  Display All Clients: View a formatted list of all registered accounts.
*  Add New Client: Register new users with unique account numbers.
*  Update Client: Modify existing client details dynamically.
*  Delete Client: Remove records safely from the file.
*  Search: Quickly find client information using account numbers.
*  Data Persistence: All data is saved in a `.txt` file, ensuring it isn't lost when the program closes.

##  Technologies & Concepts
* **Language:** C++11 or higher.
* **OOP Principles:** Encapsulation (Classes), Abstraction, and Data Management.
* **STL (Standard Template Library):** Heavy use of `std::vector` for memory management and `std::string`.
* **File I/O:** `fstream` library for reading and writing data.
